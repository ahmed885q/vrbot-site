# نظام الحماية المتقدم للبوت

## الميزات المدمجة

### 1. نظام محاكاة السلوك البشري
- حركة ماوس منحنية وغير مباشرة
- تأخيرات عشوائية بين الإجراءات
- محاكاة الأخطاء البشرية الطبيعية
- فترات راحة عشوائية
- حركات صغيرة عشوائية

### 2. نظام الحماية من الاكتشاف
- إخفاء توقيت الإجراءات
- تناوب وكلاء المستخدم
- إخفاء الأنماط السلوكية
- نظام إنذار مبكر
- وضع التخفي الطارئ
- تشويش بصمة المتصفح

## التكوين

### إعدادات السلوك البشري
```typescript
{
  minDelay: 800,        // أقل تأخير (مللي ثانية)
  maxDelay: 3000,       // أقصى تأخير (مللي ثانية)
  mouseAccuracy: 0.9,   // دقة الماوس (90%)
  errorRate: 0.05,      // معدل الأخطاء (5%)
  randomBreaks: true,   // فترات راحة عشوائية
  humanizeMouse: true   // محاكاة حركة الماوس البشرية
}
